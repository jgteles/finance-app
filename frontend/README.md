# Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. **npm** install recharts
3. npm install xlsx
4. npm install file-saver
5. Run the app:
   `npm run dev`
6. Run the on backend;

   python manage.py runserver
